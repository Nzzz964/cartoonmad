# Cartoonmad Spider

> **LINK :** [动漫疯官网](https://www.cartoonmad.com/)

## Requirement

- `python3`
  - `requests`
  - `lxml`

## Usage

```shell
source ./venv/bin/activate
python main.py

# Input comic url
输入要下载的漫画链接: https://www.cartoonmad.com/comic/3864.html
# Input which part you want
请选择要下载的章节(e.g. 1-5 1): 1-10
# Wait for download
Downloading...
Finished :)
```
